Image is used to insert and display an image within a component.

* * *

```
<Image src="https://i.imgur.com/Z7AzH2c.png" alt="Sky and roof" />
```

```
<Image
  src="https://i.imgur.com/Z7AzH2c.png"
  alt="Sky and roof"
/>
```

```
<Image
  src="https://i.imgur.com/Z7AzH2c.png"
  alt="Sky and roof"
/>
```

## Content[#](#content)

* * *

The `Image` component represents an image within a Spectrum container such as a [Dialog](https://react-spectrum.adobe.com/react-spectrum/Dialog.html). `Image` provides no specific styling by itself, but receives styling from the parent container. In addition, `Image` will be automatically placed within the container's layout according to Spectrum guidelines.

**Note: Images must have an `alt` attribute.** If the image is considered decorative and should not be announced by screen readers, then set `alt=""` to suppress the warning.

See [Dialog](https://react-spectrum.adobe.com/react-spectrum/Dialog.html#examples) for examples of how to use the `Image` component in the context of a Spectrum container.

To internationalize an `Image`, a localized string should be passed to the `alt` prop.
